# 视频分析与字段

只处理 `type=video` 的记录。

## analysis 对象

```json
{
  "viral_breakdown": "爆款拆解",
  "reusable_tactics": "可复制部分"
}
```

## 分析要求

`viral_breakdown` 必须覆盖开场钩子、痛点、叙事顺序、口播结构、证据和 CTA。`reusable_tactics` 抽象成可复用的脚本公式，不照搬来源内容。

只有字幕和元数据时，使用“字幕显示”“从口播结构看”“适合配合”等谨慎表述。未检查视频帧时，不得写“画面展示了”。

字幕经过高置信度纠错后写入历史字段 `口播文案（原始字幕）`。常见纠错包括：

```text
codes/codax/codesi -> Codex
s q/sq -> Skill
华人勋 -> 黄仁勋
mac档 -> Markdown
卡巴西 -> 卡帕西
入手 -> 助手
外科手术师修改 -> 外科手术式修改
```

不确定的词保留原文，不擅自润色说话风格。无字幕时写 `未获取字幕`，继续分析可确认的元数据。

## 飞书字段顺序

```text
标题
链接
封面
作者
视频时长
点赞数
收藏数
评论数
分享数
正文
口播文案（原始字幕）
爆款拆解
可复制部分
```

点赞数、收藏数、评论数、分享数为数字。
