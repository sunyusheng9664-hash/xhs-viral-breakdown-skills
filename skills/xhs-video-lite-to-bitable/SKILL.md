# 小红书视频采集归档 Skill Lite

把用户主动提供的公开小红书视频链接采集为基础素材记录。Lite 版只做公开视频元数据采集、备份、去重和飞书归档，不获取字幕，不生成爆款拆解，不总结可复制脚本。

## 固定流程

1. 将本文件所在目录记为 `<skill-dir>`。所有命令都必须使用 `<skill-dir>/scripts/xhs-video-lite.mjs` 的绝对路径，不得假设当前工作目录是 Skill 目录。
2. 先运行：

   ```text
   node "<skill-dir>/scripts/xhs-video-lite.mjs" doctor
   ```

3. 若 Lite 配置不存在，读取 [setup-and-archive.md](references/setup-and-archive.md)：
   - 优先从 Pro 版配置迁移视频库绑定；
   - 或绑定用户提供的已有视频库；
   - 只有用户明确同意后，才创建一个 `小红书视频采集试用库`。
4. 把用户原始输入保存到临时文本文件，运行：

   ```text
   node "<skill-dir>/scripts/xhs-video-lite.mjs" extract --input <输入文件> --output <采集结果.json>
   ```

5. 不下载字幕，不填写 `analysis` 对象，不补充爆款分析字段。
6. 运行归档：

   ```text
   node "<skill-dir>/scripts/xhs-video-lite.mjs" archive --input <采集结果.json> --output-dir <备份目录>
   ```

7. 返回检测链接数、视频成功数、图文或失败链接原因、写入数、重复数、飞书链接和 Excel 备份路径。

## 字段边界

Lite 版只写入：

```text
标题
链接
封面
作者
视频时长
点赞数
收藏数
评论数
分享数
正文
采集时间
失败原因
```

不得新增或填写以下付费版字段：

```text
口播文案（原始字幕）
爆款拆解
可复制部分
封面分析
互动诱因
爆款原因
可复制点
```

## 硬性边界

- 只处理公开视频笔记，不登录小红书、不绕过权限、不批量搜索。
- 遇到图文笔记，记录失败原因并提示使用基础版图文 Skill 或 Pro 版统一 Skill。
- 不得编造标题、正文、互动数、作者、时长、封面或未看到的画面。
- 不得获取字幕、逐字稿、口播文案或进行爆款分析。
- 必须先生成 Excel 备份，再尝试写飞书。
- 飞书失败时交付备份和可操作的失败原因。
- 不得把飞书凭证、Base token 或用户内容写入 Skill 文件。
- 不得新建飞书库，除非用户在当前对话中明确同意。

## 完成标准

- 所有输入链接都有成功或失败结论。
- 成功项只包含视频公开基础数据。
- 数字字段保持数字类型。
- 重复运行同一链接不新增记录。
- 返回可打开的 `.xlsx` 备份。
