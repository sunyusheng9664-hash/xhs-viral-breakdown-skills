## v1.2.0 - 图片长期归档与安全升级

### 解决什么问题

- 图片链接会失效：现在保存真实附件，不再只存临时链接。
- 老用户怕升级破坏表格：先预览差异，确认后只新增字段，不删除旧数据。
- 历史图片坏了难修：可根据原笔记链接重新获取并补传图片。
- 失败原因看不懂：会区分抓取受限、短链失效、下载失败、上传失败和部分成功。

### 新增能力

- 图文：首图存“封面图片”，全部图片按顺序存“图片附件”。
- 视频：封面存“封面图片”。
- 新增 Schema v2 迁移预览、确认迁移和迁移报告。
- 新增历史图片修复预览、分批修复和结果报告。
- 重复执行不会重复创建字段；修复时只补同名缺失附件。

### 回滚

升级前标签：`pre-v1.2.0-20260714`。发布标签：`v1.2.0`。

回滚代码不会自动删除飞书新增字段或附件；它们可继续保留，不影响旧字段。
