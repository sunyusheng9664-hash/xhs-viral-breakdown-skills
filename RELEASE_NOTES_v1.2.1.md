# v1.2.1 更新说明

这次优化主要解决一个问题：旧用户更新 Skill 后，不应该在不知情的情况下直接改原来的飞书多维表格。

## 做了什么

- 自动区分“首次安装”“旧版更新”和“已经升级”。首次安装不会误弹升级提示。
- 旧版更新时，先向用户说明本次更新内容、原表需要怎样调整，以及准备如何执行。
- 用户明确同意后才新增缺失字段并调整默认视图；旧字段、旧数据和自定义字段不会删除。
- 每次方案都有唯一 `plan_id`。如果检查后表结构发生变化，原授权自动失效，需要重新确认。
- 表结构升级与历史图片修复分开授权，不能一次同意后连续执行两类操作。

## 对小白意味着什么

更新后你会先看到一段清楚的改造说明。只有回复同意，Skill 才会升级原表；不同意或暂不处理时，原表不会被修改。

## 回滚

- 升级前代码：`pre-v1.2.1-20260714` 或 `v1.2.0`
- 本版本：`v1.2.1`

回滚代码不会自动删除已经新增到飞书表格中的字段；这些字段可以保留，不影响旧版继续读取原字段。
